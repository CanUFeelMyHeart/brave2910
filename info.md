## Команда git clone:
>Позволяет перенести удалённый репозиторий в vs code и работать с ним.

## Команда git push:
>Позволяет отправлять на GitHub изменения произведённые в vs code.

### Команда git pull
>Позволяет принимать изменения из GitHub в vs code
